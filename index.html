My 8-month co-op at PNI was centered around an ongoing project designing and implementing a system for device recognition and tracking using read-write RFID transponders. As the lead on this project, I personally:

  • Sourced a compact ISO-15693 compliant RFID read-write module and gamma-tolerant transponders with programmable FRAM user memory
  • Created a specification for RFID transponder memory layout, to store critical device information and usage logs with provisions for data reduncancy
  • Developed a software library to interface the RFID module with a microcrontroller via RS-232 serial connection
  • Built a touch-screen device for programming custom information onto the RFID transponders quickly and efficiently

Other responsibilities included regular microfluidic device fabrication and testing, editing of device manuals, and assisting with instrument builds.
